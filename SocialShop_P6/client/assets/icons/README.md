# PWA Icons

Đặt 2 file icon vào thư mục này:
- `icon-192.png` — 192x192 px
- `icon-512.png` — 512x512 px

## Tạo icon nhanh (online)
1. Vào https://favicon.io/favicon-generator/
2. Text: "SS", Background: #7C5CFC, Font color: #FFFFFF, Font: Rounded
3. Download và rename thành icon-192.png / icon-512.png
4. Hoặc dùng bất kỳ công cụ thiết kế nào (Canva, Figma, v.v.)

Màu thương hiệu SocialShop: #7C5CFC (tím gradient)
